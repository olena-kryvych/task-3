Ігри для прокачки CSS

Коротесенькі враження про пройдені мною ігри

  • Flexbox Defense

    • Цікава гра. Легко запом'ятовується інформація. Вона складається всього з 12 рівнів і це дуже добре, оскільки мені не сподобалась тематика гри зі своїм музичним супроводом, через що було бажання як можна швидше її пройти.

    Flexbox Defense
  • Flexbox Froggy

    • Завдяки Flexbox Froggy я змогла ознацомитись з основами верстки. Гра складається з 24 етапів, і всі вони дуже цікаві. При проходженні цієї гри час летить дуже швидко. Вона мені чимось нагадала мультсеріал з мого дитинства - Dora the Explorer (там потрібно було "допомогти" головній героїні дійти до призначеного місця). Це був добрий флешбек, тому мені гра дуже сподобалась.

    Flexbox Froggy
  • CSS Diner

    • CSS Diner — гра, яка навчила правильно звертатися до конкретних елементів у CSS-стилях. На перший погляд може здатися, що завдань забагато, але після прочитання умов ці думки зникають, адже завдання не дуже складні.

    CSS Diner
